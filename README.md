# Rotator-ESP32-S3-Board

![alt text](https://github.com/brokebit/Rotator-ESP32-S3-Board/blob/main/production/Controller_Rendering.jpg?raw=true)


# Features / Design Info 
- GPS Time Sync using NEMA GPS Module over RS232
- Serial interface for Nextion Display
- Designed for Clearpath servo motors from https://teknic.com/
- Interface for 6 Hardware Debounced buttons for manual direction control, homing and moon tracking 
- Uses pulse encoder data from the Clearpath motors for position feedback
- Power input is +10v - +25v DC
- Motors are powered by seperate power source 
- Voltage select headers to select either 3.3v or 5v logic for RS232 interfaces (GPS, LCD)
- 6 Layer Board
